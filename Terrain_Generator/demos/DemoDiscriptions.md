# Demo Programs 📂

这个文件夹包含多个演示程序，展示 Terrain Generator 的各种用法和参数效果。

## Demo 列表

| Demo 文件 | 名称 | 说明 |
|----------|------|------|
| `demo01_basic.cpp` | 基本地形生成 | 展示默认参数下的基本地形生成 |
| `demo02_smooth_plains.cpp` | 平滑平原 | 使用低八度和低持久度生成平滑地形 |
| `demo03_rugged_mountains.cpp` | 崎岖山地 | 使用高八度和高持久度生成崎岖地形 |
| `demo04_large_scale.cpp` | 大型地形特征 | 使用低缩放比例生成大型地形特征 |
| `demo05_small_scale.cpp` | 小型细节地形 | 使用高缩放比例生成小型细节 |
| `demo06_variable_density.cpp` | 可变密度显示 | 展示不同密度方块的显示效果 |
| `demo07_comparison.cpp` | 多地形对比 | 同时展示多种不同配置的地形 |

## 编译和运行 Demo

### 方法 1: 替换 main.cpp
将某个 demo 文件的内容复制到 `src/main.cpp` 中，然后使用 Visual Studio 编译运行。

### 方法 2: 创建单独的项目（高级）
为每个 demo 创建单独的项目配置文件（.vcxproj），这样可以同时保留多个 demo。

## 快速参考

### 参数调优速查表

| 效果 | scale | octaves | persistence |
|------|-------|---------|-------------|
| 平滑平原 | 0.01-0.03 | 2-4 | 0.3-0.5 |
| 平衡地形 | 0.02-0.04 | 4-6 | 0.5-0.6 |
| 崎岖山地 | 0.02-0.03 | 6-10 | 0.7-0.9 |
| 大型特征 | 0.005-0.015 | 4-6 | 0.5-0.6 |
| 小型细节 | 0.05-0.10 | 5-8 | 0.5-0.7 |

### 参数说明

- **scale**：控制噪声的"缩放"。值越小，地形特征越大；值越大，地形细节越多。
- **octaves**：八度数量。每个八度叠加更多细节，但会增加计算时间。
- **persistence**：持久度。控制每个八度的振幅衰减速度。值越大，高频细节越突出。
